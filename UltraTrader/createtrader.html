Create A Trader

NPC

Creating an NPC trader is very simple, follow the following instructions and you should be up and running in no time.

Create the NPC

You will need to use a Citizens 2 command to create an NPC.

/npc create npcName

Where npcName is the name of the NPC you wish to create.

Assign the UltraTrader trait

After the NPC has been created, you can now assign all the required traits.

Manually with Citizens

An administrator might want to assign traits by hand, to do that you would use the citizens trait command. After making sure you have first selected it.

/trait traitname

With UltraTrader Command

Players should use the UltraTrader command.

/trader create or /trader create group

This will assign all the required traits to the NPC that is selected and is owned by the player.

Activate the Shop

Next you are going to either assign or create a shop. This shop can be any shop you own, or have access too, or be brand new. All you need to do is right click the NPC and start a conversation with him.

Once the conversation is started all you need to do is follow the instructions and everything will be taken care of for you.

When done, just click the trader again, and your shop will open for you.

Adding your first Category

UltraTrader shops require the use of Categories, these are easily added by just dragging an item stack you want to use as the category icon on the traders main inventory screen (The screen that first opens) and following the instructions in the conversation.

Create Admin Shop

To create an admin shop, all you need to do is set the inventory type and the wallet type to admin. This can easily be done from within the Admin conversation started when you click the book and quill in either of the buy or sell screens.
